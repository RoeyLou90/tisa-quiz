# 好好證券 TISA 投資人格測驗

本專案為好好證券的TISA投資人格測驗活動頁，包含前端Vue.js與後端Node.js實作。

## 專案架構

- `frontend/` - Vue.js前端程式碼
- `backend/` - Node.js後端API服務

## 功能說明

- 投資人格測驗問卷（12題）
- 計分邏輯判定人格類型
- 結果頁面顯示人格分析與投資建議

## 技術棧

- 前端：Vue.js 3
- 後端：Node.js (Express)

## 啟動方式

### 前端
```bash
cd frontend
npm install
npm run serve
```

### 後端
```bash
cd backend
npm install
npm start
```
